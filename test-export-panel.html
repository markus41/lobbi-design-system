About This Test Page

This page demonstrates the export panel functionality that has been added to all 255 style pages in the Lobbi Design System gallery.

The export panel allows designers and developers to quickly extract design tokens in three formats: CSS custom properties, Tailwind configuration, and JSON.

How to Test the Export Panel

  1. Look at the bottom-right corner of this page - you'll see the export panel
  2. Click the CSS button to copy all design tokens as CSS custom properties
  3. Click the Tailwind button to copy tokens formatted for Tailwind CSS config
  4. Click the JSON button to copy tokens as a JSON object
  5. After clicking any button, a green toast notification will appear confirming the copy
  6. Paste the copied content into a text editor to verify the format
